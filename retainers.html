Web edit units & ticket tiers

Each website care request is classified as Light, Standard, or Heavy. Your retainer includes a monthly unit budget (for example 4 or 8 units). Each ticket spends units: Light = 1 edit unit, Standard = 2 edit units, Heavy = 3 edit units. You can mix tiers any way you like within that monthly total — e.g. four Light tickets use all 4 units; one Heavy plus one Light uses 3 + 1 = 4 edit units.

Scoped updates (site foundation update, additional page, premium landing/hero — under Web Services → Website care) are not counted in ticket units. Essential web and Full Digital Essential include 1 scoped update per month; Growth plans include 2 per month, as stated in the rows above.
